EcoWiFi Assembly Guide

Step 1 of 8

Build Your EcoWiFi System

Follow this comprehensive step-by-step guide to assemble your EcoWiFi system that converts plastic bottle recycling into free internet access.

1

Hardware Setup

Components Needed:

  • Orange Pi One (Main Controller)
  • Mikrotik Router (Network Gateway)
  • ESP32 (Sensor Controller)
  • USR-ES1 W5500 Ethernet Module

Actions:

  1. 1. Flash Armbian to Orange Pi One SD card
  2. 2. Connect all components to power
  3. 3. Verify LED indicators on all devices
  4. 4. Connect Orange Pi 4 to network via Ethernet

Tip: Use a surge protector for all power connections to prevent damage during setup.

2

Sensor Installation

HC-SR04 Ultrasonic
  • • Top: Bottle detection
  • • Bottom: Bin level monitoring
  • • GPIO 5, 18 (Trigger, Echo)
MG996R Servo Motor
  • • Bottle sorting mechanism
  • • GPIO 21 (PWM control)
  • • External power required
AS7265x Spectroscopy
  • • Plastic material detection
  • • Qwiic I2C interface
  • • SDA: GPIO 22, SCL: GPIO 23

Important: Calibrate each sensor before final installation. Test with sample bottles to ensure proper detection.

3

Network Configuration

Mikrotik Router Setup:

WiFi Network: EcoWiFi-Guest

Security: WPA2-PSK

Captive Portal: Enabled

Session Timeout: 15 minutes

IP Address Configuration:

Orange Pi 4: 192.168.1.10

Mikrotik: 192.168.1.1

ESP32/W5500: 192.168.1.20

Gateway: 192.168.1.1

4

Software Installation

Install Bun Runtime:

curl -fsSL https://bun.sh/install | bash

Setup Project:

bun init ecowifi-server
cd ecowifi-server
bun add hono better-sqlite3 @hono/node-server

Initialize Database:

bun run src/migrate.ts
5

ESP32 Programming

Arduino IDE Setup:

  • 1. Install ESP32 Board Manager
  • 2. Select "ESP32 Dev Module"
  • 3. Upload sensor control code
  • 4. Test sensor readings

Key Libraries:

  • • WiFi.h (Network connectivity)
  • • Ethernet.h (W5500 support)
  • • HX711.h (Load cell)
  • • NewPing.h (Ultrasonic)
  • • Wire.h (I2C for AS7265x)
6

Mechanical Assembly

Enclosure Construction:

  • • Weatherproof housing for outdoor installation
  • • Bottle insertion slot (3-5 cm diameter)
  • • Secure mounting brackets for sensors
  • • Proper ventilation for electronics

Sensor Placement:

  • • Top ultrasonic: 10cm above insertion slot
  • • Bottom ultrasonic: 5cm above bin base
  • • AS7265x: Inside bottle path for material scanning
  • • Servo motor: Connected to sorting mechanism
