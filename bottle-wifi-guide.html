EcoWiFi System Guide

Internet Access for Plastic Bottles

System Overview

How It Works

  1. 1
    Connect to WiFi: Users connect to the EcoWiFi network
  2. 2
    Access Payment Page: Captive portal redirects to bottle deposit interface
  3. 3
    Deposit Bottle: Insert plastic bottle into the recycling slot
  4. 4
    Get Internet Access: Receive 15 minutes of WiFi connectivity

Key Features

  • Automated bottle detection
  • 15-minute WiFi access per bottle
  • Real-time system monitoring
  • Secure captive portal system
  • Environmental impact tracking

Hardware Components

Orange Pi One

Main processing unit, runs Bun/Hono web server with SQLite database

Key Role: System Controller

Mikrotik Router

Network management, captive portal, WiFi access point

Key Role: Network Gateway

USR-ES1 W5500

Ethernet module for hardware SPI to LAN connectivity

Key Role: Network Interface

ESP32

Controls sensors, manages bottle detection and system triggers

Key Role: Sensor Controller

Technical Specifications

Network Configuration

WiFi Network: EcoWiFi-Guest
Authentication: Captive Portal
Session Duration: 15 minutes
Bandwidth Limit: 2 Mbps per user
Max Users: 20 concurrent

Software Stack

  • Bun Runtime: High-performance JavaScript runtime
  • Hono Framework: Fast web framework
  • SQLite: Lightweight embedded database
  • WebSocket: Real-time communication
